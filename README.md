## Setup

1. Clone the repository:
    git clone https://github.com/GDewade/Gen_AI.git

2. Install dependencies:
    pip install -r requirements.txt
   
4. Create a .env file and add your API keys:
    OPENAI_API_KEY=your_api_key_here
